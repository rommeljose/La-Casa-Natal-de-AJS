Párrafo

A partir del original (2015), este libro fué reescrito en XML 1  (eXtensible Markup Language); un meta-lenguaje que permite definir lenguajes de marcas; desarrollado por el World Wide Web Consortium (W3C 2 ). Utilizando Notepad ++ 3  un editor de código fuente (Software Libre; licencia GPL), y editado para su publicación con PreTeXt 4 ; un vocabulario XML para documentos académicos; con PreTeXt se puede describir el contenido y la estructura de un libro de texto, monografía o artículo académico (Software Libre; licencia GPL). Se utilizó como sistema opertivo el CygWin 5 , una colección de herramientas GNU -de código abierto- que proporcionan una funcionalidad similar a una distribución de Linux bajo Windows (Software Libre; licencia GPL y LGPL).
